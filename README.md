A Flask web application that uses a fine-tuned BERT language model to detect hate speech in Kannada text.

## Features
- User Signup / Login
- OTP Email Verification
- Hate Speech Detection using BERT
- Clean UI + Mobile Responsive
- Dashboard with Hate/Non-Hate examples
- SQLite database

## Tech Stack
- Python Flask
- SQLite
- Transformers (BERT)
- Torch
- HTML/CSS
- Render (Hosting)

## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python main.py

3. Visit:
   http://localhost:5000
